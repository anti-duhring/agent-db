**Package:** `package dynamodb`

**Constants and key encoding functions (per D-02, D-03):**
- `tableName = "chat_data"`
- `userPK(partnerID, userID uuid.UUID) string` returns `"USER#<partnerID>#<userID>"`
- `convSK(updatedAt time.Time, convID uuid.UUID) string` returns `"CONV#<updatedAt.UTC().Format(time.RFC3339Nano)>#<convID>"`
- `convPK(convID uuid.UUID) string` returns `"CONV#<convID>"`
- `msgSK(createdAt time.Time, msgID uuid.UUID) string` returns `"MSG#<createdAt.UTC().Format(time.RFC3339Nano)>#<msgID>"`

**Record types for attributevalue marshaling (use `dynamodbav` struct tags):**
- `convMetaRecord`: PK=`CONV#<conv_id>`, SK=`CONV#META`, stores `conv_id`, `partner_id`, `user_id`, `created_at`, `updated_at`, `user_pk` (the USER# PK string), `conv_listing_sk` (current listing SK for delete in AppendMessage)
- `convListingRecord`: PK=`USER#<partner>#<user>`, SK=`CONV#<updated_at>#<conv_id>`, stores `conv_id`, `partner_id`, `user_id`, `created_at`, `updated_at`
- `msgRecord`: PK=`CONV#<conv_id>`, SK=`MSG#<created_at>#<msg_id>`, stores `msg_id`, `conversation_id`, `role`, `content`, `token_count`, `created_at`

**CreateConversation (per D-01, D-02):**
- Generate `id := uuid.New()`, `now := time.Now().UTC()`
- Build listing SK: `convSK(now, id)`
- Use `TransactWriteItems` with 2 Puts:
  1. Put convMetaRecord at PK=`CONV#<id>`, SK=`CONV#META` with `user_pk`, `conv_listing_sk`, `partner_id`, `user_id`, `created_at`, `updated_at`
  2. Put convListingRecord at PK=`USER#<partner>#<user>`, SK=`CONV#<now>#<id>` with `conv_id`, `partner_id`, `user_id`, `created_at`, `updated_at`
- Return `domain.Conversation{ID: id, PartnerID: partnerID, UserID: userID, CreatedAt: now, UpdatedAt: now}`

**AppendMessage (per D-04 — TransactWriteItems, 3 items):**
- Generate `id := uuid.New()`, `now := time.Now().UTC()`, `tc := len(content) / 4`
- GetItem on PK=`CONV#<conversationID>`, SK=`CONV#META` with `ConsistentRead: true` to retrieve `user_pk` and `conv_listing_sk` (the old SK to delete)
- Build new listing SK: `newSK := convSK(now, conversationID)`
- `TransactWriteItems` with 3 items:
  1. Put msgRecord at PK=`CONV#<conversationID>`, SK=`MSG#<now>#<id>`
  2. Delete old listing item at PK=`<user_pk>`, SK=`<old_conv_listing_sk>`
  3. Put new convListingRecord at PK=`<user_pk>`, SK=`<newSK>` with updated timestamp
- Also update the convMetaRecord's `conv_listing_sk` and `updated_at` — add a 4th item: Put convMetaRecord with new `conv_listing_sk=newSK` and `updated_at=now`
- Return `domain.Message{ID: id, ConversationID: conversationID, Role: role, Content: content, TokenCount: tc, CreatedAt: now}`

**LoadWindow (per D-03, D-05, D-06):**
- Use expression builder: `expression.Key("PK").Equal(expression.Value(convPK(conversationID)))`
- Add SK begins_with condition: `expression.Key("SK").BeginsWith("MSG#")`
- Query with `ScanIndexForward: aws.Bool(false)`, `Limit: aws.Int32(int32(n))`, `ConsistentRead: aws.Bool(true)`
- Parse each item: unmarshal into msgRecord, parse `created_at` with `time.Parse(time.RFC3339Nano, ...)`, parse UUIDs
- Reverse the slice in-place (same as Postgres adapter) for oldest-first order
- Return `[]domain.Message` (empty slice not nil when no results)

**ListConversations (per D-02, D-05, D-06):**
- Use expression builder: `expression.Key("PK").Equal(expression.Value(userPK(partnerID, userID)))`
- Query with `ScanIndexForward: aws.Bool(false)`, `ConsistentRead: aws.Bool(true)`
- Parse each item: unmarshal into convListingRecord, parse timestamps and UUIDs
- Return `[]domain.Conversation{}` (empty slice not nil)

```go
func setupLocalStack(t *testing.T) *DynamoDBRepository {
    t.Helper()
    ctx := context.Background()
    ctr, err := localstack.Run(ctx, "localstack/localstack:latest")
    require.NoError(t, err)
    t.Cleanup(func() { testcontainers.TerminateContainer(ctr) })

    // Get the DynamoDB endpoint from LocalStack
    host, err := ctr.Host(ctx)
    require.NoError(t, err)
    port, err := ctr.MappedPort(ctx, "4566/tcp")
    require.NoError(t, err)
    endpoint := fmt.Sprintf("http://%s:%s", host, port.Port())

    repo, err := New(ctx, endpoint)
    require.NoError(t, err)
    return repo
}
```
